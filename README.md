# gulp4 / Animation hell box

### Подкачать зависимости
`nmp i`

### Собрать сборку   
`gulp build`


### Запустить  
`gulp`


### Удалить все изображения   
`gulp cleanimg`


### Генерировать сетку SmartGrid Дмитрия Лаврика   
`gulp grid`


### Генерировать Критические и Асинхронные стили
`gulp criticalStart`
